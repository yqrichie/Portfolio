The Flow of Time

时间的流淌

在Quora上有人提问“什么是时间”，其中一个人的回答是：All thought is either a memory or an imagination. Any idealization or conceptualization is always about objects. Can there be objects without time? If nothing changes at all, would there be time? So if there is no movement in the mind, is there time? So, all movement is time. All activity of the mind is “time”.
设计灵感来源于苹果公司曾经的一部点线MG动画宣传片，以简单的点、线、面构成了多样的变化。 我的这部MG动画中大量地使用了粒子插件和Newton插件来制作丰富的动画效果。在声音的处理上，加入了混响营造氛围，配合Panaroma插件制作3D立体声效，佩戴耳机观看时左右声道的变换会让观者体验声音的空间感，感受时间的流逝。
